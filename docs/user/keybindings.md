# Keybindings

GNOME Activities does not set keybindings automatically. You can configure
custom keyboard shortcuts in GNOME Settings > Keyboard > Custom Shortcuts.

## Recommended Shortcuts

| Action | Suggested Shortcut | Command |
|--------|-------------------|---------|
| Switch to Work activity | `Super+1` | `gnome-activities activate Work` |
| Switch to Personal activity | `Super+2` | `gnome-activities activate Personal` |
| Show current activity | `Super+A` | `gnome-activities current` |

## Adding Custom Shortcuts in GNOME

1. Open **Settings** → **Keyboard** → **View and Customize Shortcuts**
2. Scroll to **Custom Shortcuts**
3. Click **+** to add a new shortcut
4. Set the **Name** (e.g., "Switch to Work")
5. Set the **Command** (e.g., `gnome-activities activate Work`)
6. Set the **Shortcut** (e.g., `Super+1`)
